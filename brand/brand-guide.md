# Waseet CRM — Brand Guide

## Name
- Arabic: **وسيط** (Waseet — means "broker")
- English: **Waseet CRM**
- Tagline (Arabic): **إدارة عملاء العقار، بسهولة**
- Tagline (English): **Real Estate CRM, the Arabic Way**

## Typography

**Arabic Font:** Cairo (Google Fonts — free)
- Import: `https://fonts.googleapis.com/css2?family=Cairo:wght@400;600;700;900&display=swap`
- Use for all Arabic text, headings, UI labels

**English / Numbers Font:** Inter (Google Fonts — free)
- Import: `https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap`
- Use for English text, numbers, SAR amounts

**Font Sizes:**
- Hero heading: 48-56px
- Section heading: 32px
- Subheading: 22-24px
- Body: 16-18px
- Label/caption: 14px

## Voice & Tone

- Direct and confident (not corporate)
- Written in Arabic first
- Addresses the agent as a professional: "أنت الوسيط" (You are the broker)
- No jargon — simple language that a non-tech agent understands
- Emphasize outcomes: more deals, less chaos, save time
